# ZLineBot Roadmap (EN)

## Phase 1 (0-30 days)
Stabilize APIs, improve docs, strengthen auth, and improve admin UX.

## Phase 2 (30-60 days)
Improve payment reliability, webhook idempotency, and compliance SLA workflows.

## Phase 3 (60-120 days)
Deploy production personalization/ranking and measurable experimentation.

## Phase 4 (120-180 days)
Enterprise scale: multi-region, SSO/SCIM, partner API readiness.

## Cross-cutting tech debt
- Distributed rate limiting
- Schema migration framework
- CI test pyramid
- Backend/admin feature parity
